## 주제 1 — 기상 조건에 따른 타슈 대여 패턴 분석

> **"비 오는 날 정말 따슈를 안 탈까?"**

대전 기상청 ASOS 분별 관측 데이터와 타슈 대여이력을 결합하여, 강수·폭염 등 기상 조건이 자전거 이용에 미치는 영향을 분석합니다.

### 데이터

| 데이터 | 출처 | 기간 | 건수 |
|--------|------|------|------|
| 타슈 대여이력 | 공공데이터포털 (대전교통공사) | 2026년 1~3월 | 1,319,762건 |
| ASOS 분별 기상 | 기상청 기상자료개방포털 (지점 133) | 2026년 1~3월 | 129,594건 |

### 핵심 쿼리

- **쿼리 1-A** — 강수량 조건별(없음 / 10mm 미만 / 10mm 이상) 시간당 평균 대여량 비교 `LEFT JOIN + GROUP BY`
- **쿼리 1-B** — 체감온도 35°C 이상 폭염 기간 대여량 급감 대여소 상위 5곳 `CTE + JOIN + GROUP BY`

### DB 테이블

```
rental_station   대여소 마스터 (station_id PK)
weather          ASOS 시간별 기상 관측 (obs_datetime, precipitation, feel_temp …)
rental           타슈 대여이력 (rental_station_id FK → rental_station)
```

## 주제 2 — 국제 유가와 국내 주유소 가격의 시차 상관관계 분석

> **"국제 유가가 오르면 우리 동네 기름값은 언제 오를까?"**

한국석유공사(오피넷)의 국제 유가 데이터와 대전시 주유소 일별 판매가격을 결합하여, 유가 변동이 국내 판매가에 반영되는 시차(Lag)와 브랜드별 반응 속도를 분석합니다.

### 데이터

| 데이터 | 출처 | 기간 | 건수 |
|--------|------|------|------|
| 국제 원유 가격 (Dubai·Brent·WTI) | 한국석유공사 오피넷 | 2023.01.03 ~ 2026.06.04 | 883건 |
| 대전시 주유소 일별 판매가격 | 공공데이터포털 오피넷 API | 2026.05.07 ~ 2026.06.06 | 6,101건 |

### 핵심 쿼리

- **쿼리 2-A** — 브랜드별 유가 상승 후 평균 반응 일수 분석 `LAG 윈도우 함수 + JOIN + GROUP BY`
- **쿼리 2-B** — 유가 상승기·하강기별 가격 변동 비대칭성 검증 (로켓과 깃털) `이동평균 윈도우 함수 + CASE + GROUP BY`

### DB 테이블

```
oil_price    국제 유가 (price_date PK, dubai, brent, wti)
gas_station  주유소 마스터 (station_id PK, brand)
gas_price    주유소 일별 판매가격 (station_id FK → gas_station)
```

## 실행 방법

Python 파일은 표준 라이브러리(`sqlite3`, `csv`, `zipfile`)만 사용하므로 별도 설치 없이 실행 가능합니다.

```bash
# 주제 1: 타슈 + 기상 분석
python 01-타슈_날씨_분석.py

# 주제 2: 국제 유가 + 주유소 분석
python 02-국제유가_시차분석.py
```

각 스크립트 실행 시 SQLite DB 파일(`tashu_weather.db`, `oil_gas.db`)이 생성되고, 쿼리 결과가 터미널에 출력됩니다.
